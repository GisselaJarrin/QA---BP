CONCLUSIONES - Proyecto de Pruebas E2E con Cypress

2. La estructura del proyecto permite realizar pruebas organizadas, modulares y reutilizables.

3. Se puede trabajar fácilmente con APIs REST usando `cy.request()` para validar respuestas de forma rápida.

5. Cypress también facilita el trabajo en equipo gracias a su interfaz visual, donde se pueden revisar resultados de manera clara.

6. Simular flujos reales de usuario (registro + login) y validar respuestas HTTP con aserciones.
